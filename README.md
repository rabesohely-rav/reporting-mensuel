## Usage
Cette version est :
- compatible GitHub Pages
- sans serveur
- sans Node.js
- avec sauvegarde dans le `localStorage`

## Fonctionnalités
- page 1 : formulaire
- pages 2 et 3 : impression A4 portrait
- impression limitée aux pages 2 et 3
- sommes automatiques
- pourcentages sur les graphiques
- logos intégrés
