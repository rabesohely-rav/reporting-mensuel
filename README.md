# Reporting mensuel France Services — version localStorage

## Ce package est le bon choix si tu veux du simple

- aucun serveur
- aucun Node.js
- aucune base de données
- sauvegarde dans le navigateur avec `localStorage`
- compatible hébergement statique
- impression PDF via le navigateur

## Déploiement

Tu peux :
- ouvrir `index.html` directement dans le navigateur
- ou publier le dossier sur GitHub Pages

## Limite

Les données restent liées au navigateur utilisé.  
Elles ne sont pas synchronisées entre plusieurs postes.
