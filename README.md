# Reporting mensuel France Services — package GitHub-ready

## Cette version est faite pour GitHub Pages

- aucune base de données
- aucun serveur
- aucune dépendance Node.js
- sauvegarde simple dans le navigateur avec `localStorage`
- impression limitée aux pages 2 et 3 en A4 portrait
- logos inclus dans le dépôt

## Ce qui a été corrigé

- somme automatique de « commune de l’usager »
- impression des seules pages 2 et 3
- mise en page A4 portrait plus stable
- logos France Services et Carcassonne Agglo aux emplacements fixes
- étiquettes en % sur les graphiques

## Déploiement

Dépose ces fichiers à la racine du dépôt GitHub, puis active GitHub Pages.

## Important

Le `localStorage` conserve les données uniquement dans le navigateur utilisé.
